## What Is Ecosystem Intelligence?

### The core insight

Developer activity predicts price. Not always immediately — but consistently, when measured correctly.

When a blockchain's GitHub commit velocity spikes, when package downloads for its SDK libraries accelerate, when new developers start building on it — those are leading indicators that price follows. Traditional crypto analytics miss this entirely.

BotIndex's ecosystem intelligence layer tracks:

- **GitHub commit velocity** — are core contributors actively building?
- **Package downloads** — are developers integrating this chain's libraries?
- **Developer wallet activity** — are builders deploying new contracts?
- **Ecosystem momentum score** — a composite signal combining all of the above

## How Does BotIndex Generate Predictions?

### Step 1: Data ingestion
BotIndex ingests real-time data from GitHub, npm, PyPI, and on-chain sources for each of the 26 supported assets.

### Step 2: Signal generation
The AI model identifies anomalies — spikes or drops in ecosystem activity that historically correlate with price movement within a 7–30 day window.

### Step 3: Signal confidence scoring
Each prediction carries a confidence score. High-confidence signals are flagged for Sentinel subscribers. Lower-confidence signals are still surfaced but with appropriate caveats.

### Step 4: Resolution tracking
BotIndex tracks every signal to resolution. This is what produces the verified 59.8% accuracy rate across 338 resolved predictions — not backtested, not cherry-picked.

## Why Ecosystem Intelligence and Not Price Charts?

Price charts are lagging indicators. They show you what already happened.

Ecosystem metrics are leading indicators. Developer activity spikes happen before price spikes — because builders commit code before users arrive, and users arrive before price moves.

BotIndex is built on this thesis. 338 resolved signals later, it holds.
